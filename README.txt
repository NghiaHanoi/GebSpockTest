README
This is customized simple project for using Geb with Maven.
The original source was downloading from Geb website.
This test script will check if Echo Servlet of Wiperdog could be response 
correctly or not.

NOTICE
We need to set title for return page of Echo Servlet, please replace the old file 
WiperdogEchoServlet.groovy of Wiperdog with the new one in this example.
